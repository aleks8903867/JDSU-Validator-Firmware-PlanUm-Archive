# JDSU-Validator-Firmware-PlanUm-Archive
The ultimate preservation archive for JDSU / Test-Um Validator NT900 &amp; ValidatorPRO. Includes final v2.91 firmware, Plan-Um v3.0.4 PC software, and manuals.

# 🛠️ JDSU / Test-Um Validator & ValidatorPRO Complete Software Archive

This repository serves as a permanent **hardware preservation archive** for the legacy network testers originally developed by Test-Um Inc. and later acquired by JDSU (now VIAVI Solutions). 

Since the official Acterna FTP servers (`ftp.germany.acterna.com`) and JDSU custom app portals (`custom-apps.jdsu.com`) were taken offline, these files became completely lost to the public internet. This complete package was recovered thanks to the community to prevent fully functional hardware from becoming e-waste.

## 📦 What is Included (The Complete Package)

This archive contains everything needed to restore, update, and fully utilize both the standard and PRO versions of the Validator testers:

### 1. PC Companion Software
* **Plan-Um v3.0.4 (Final Release):** The essential Windows PC software required to create `.job` files, layout floor plans, and manage test reports. Without this software, generating complex site layouts is impossible.

### 2. Validator (NT900 / NT950 / NT955) Firmware
* **v2.91 Firmware update (`VAL-FULL-2010Sep10-0855-VALAPP-2.91.TUI`):** The absolute final update. **Crucial fix:** Resolves the severe UI freezing bug when attempting to plot frequency graphs on older versions (like v2.32). Includes updated Agere PHY microcode (1.3).
* **v2.80 Firmware update (`VAL-FULL-2010Jan28-1254-valapp-2.80.TUI`):** Included as a fallback/stepping-stone update if your device refuses to jump straight to v2.91.

### 3. ValidatorPRO (NT1150 / NT1155) Firmware
* Includes legacy `.tui` update files for the PRO series (Optical/Fiber enabled models).

### 4. Official Documentation
* Original PDF update manuals and HTML instructions rescued from the old JDSU web portal.

## ⚠️ How to Update Firmware
1. Ensure your Validator battery is fully charged.
2. Place the `.tui` file onto the root of the Validator's Compact Flash (CF) card.
3. Boot the device, go to **Properties -> Support Info -> Manage Firmware**.
4. Select the update file. **Do not proceed unless all compatibility checks show a GREEN checkmark.**
5. The device will reboot automatically. Do not interrupt the process.
